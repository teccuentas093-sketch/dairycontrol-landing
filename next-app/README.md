Next.js + Tailwind scaffold for DairyControl

Quick start

1. cd next-app
2. npm install
3. npm run dev

Open http://localhost:3000

Notes:
- This scaffold is minimal: the `pages/api/chat` endpoint is a placeholder that returns simple advice.
- The hero uses `/Dairy1.mp4` and `/Dairy1-poster.svg` from the repo `public/` folder. Keep those files at the repo root `public/` so Next can serve them in development.
